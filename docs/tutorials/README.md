# Rust Tutorials

Welcome to the Rust Tutorials guide! This collection covers the essential concepts and tools you need to start developing in Rust. Whether you're new to Rust or looking to strengthen your fundamentals, these tutorials will guide you through setup, project management, naming conventions, and best practices.

## Prerequisites

- A computer with Windows, macOS, or Linux
- A text editor or IDE (Visual Studio Code recommended)
- Basic command-line familiarity

## Learning Path

We recommend following these tutorials in order:

1. **[Getting Started with Rust](getting_started.md)** — Install Rust, rustup, and set up your development environment
2. **[Cargo and rustup](cargo_and_rustup.md)** — Understand Rust's build system and toolchain manager
3. **[Naming Conventions](naming_conventions.md)** — Learn Rust's community standards for clean, readable code
4. **[Running a Rust Project](running_a_project.md)** — Build, test, and execute your first projects

## Quick Reference

After setup, the most common commands you'll use:

- `cargo new <project>` — Create a new project
- `cargo run` — Compile and run your project
- `cargo build --release` — Build an optimized release version
- `cargo test` — Run tests
