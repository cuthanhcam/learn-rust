# Using Structs to Structure Related Data

This chapter covers how to define, instantiate, and use structs in Rust, including methods and associated functions.

## Learning Goals

- model domain data using named structs
- use tuple and unit-like structs when appropriate
- create instances with shorthand and update syntax
- refactor free functions into struct methods
- distinguish methods from associated functions
