## 📖 Overview

This repository documents my journey of learning **Rust** from the ground up, with the goal of rebuilding core software engineering knowledge and developing a deeper understanding of systems.

Instead of just learning syntax, this project focuses on:

* Memory management & ownership
* Data structures & algorithms
* Concurrency and system-level thinking
* Writing safe, efficient, and maintainable code

---

## 🧠 Learning Philosophy

This repository follows a structured learning loop:

> **Learn → Code → Write → Apply → Build**

* **Learn** from high-quality resources
* **Code** concepts in isolation
* **Write** distilled knowledge in my own words
* **Apply** through problem solving
* **Build** real-world projects
