# Using Structs to Structure Related Data

This chapter explains how structs model related data with clear field names and how `impl` blocks attach behavior to those types.

Structs are a key step from simple values to domain modeling in Rust.

## Learning Goals

- define named, tuple, and unit-like structs
- instantiate and update structs safely
- understand ownership behavior during struct updates
- refactor free functions into methods
- use associated functions as constructors
